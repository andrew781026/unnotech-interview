## 開發環境

- 安裝 Node.js 14.17.0 以上

執行下述指令來啟動專案進行開發

```bash
# 使用 SSH 下載專案資料
~$ git clone git@github.com:andrew781026/unnotech-interview.git
~$ cd front
~$ npm i
~$ npm run dev
```
